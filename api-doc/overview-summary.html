Overview

Ceylan-Seaplus control C or C++ code from Erlang, not as NIF but thanks to a port, and to streamline the corresponding integration process.

This website presents the documentation of the API (Application Programming Interface) of Seaplus version 1.1.5 (as of Sunday, May 23, 2021).

Note though that Seaplus is more like a small framework than like a library, and thus users should refer primarily to its main documentation.

Ceylan-Seaplus relies on the Myriad layer, a generic Erlang toolbox.
